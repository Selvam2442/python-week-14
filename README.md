# 💳 Credit Card Fraud Detection System

## 🚀 Project Overview

The Credit Card Fraud Detection System is a Machine Learning and Data Analytics project developed to identify fraudulent credit card transactions using historical transaction data. The project combines data preprocessing, exploratory data analysis (EDA), predictive modeling, and interactive visualizations to uncover transaction patterns and detect potential fraud.

Using the Kaggle Credit Card Fraud Detection Dataset, this project demonstrates how data-driven techniques can enhance financial security, reduce fraudulent activities, and support informed decision-making in the banking and finance sector.

---

## 🎯 Objectives

- Analyze credit card transaction data.
- Perform data cleaning and preprocessing.
- Explore transaction patterns through EDA.
- Identify characteristics associated with fraudulent transactions.
- Build machine learning classification models for fraud detection.
- Evaluate model performance using classification metrics.
- Create interactive visualizations and dashboards.
- Generate actionable insights for fraud prevention.

---

## 📂 Dataset

**Source:** Kaggle – Credit Card Fraud Detection Dataset

### Features Used

| Feature | Description |
|----------|-------------|
| Time | Seconds elapsed between transactions |
| Amount | Transaction amount |
| V1-V28 | PCA-transformed anonymized features |
| Class | Target variable (0 = Legitimate, 1 = Fraudulent) |
